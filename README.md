# Research Repository

## Purpose
This repository contains all research and analysis materials used for the System Design Project.

## Contents Structure

## Contents Structure

```text
research/
│
├── Literature-Review/
├── Competitive-Analysis/
├── Client-Interviews/
```

## Included Materials
- Literature review notes
- Competitive analysis reports
- Client interview transcripts
- Research references and articles

## Tools Required
- Microsoft Word / Google Docs
- PDF Reader
- Spreadsheet Software

## Branch Strategy
- `main` → Approved research materials
- `draft/<research-topic>` → Ongoing research work
- `fix/<issue-number>` → Corrections and updates

## Contribution Guidelines
All team members must follow the contribution and branching rules defined in CONTRIBUTING.md.

## Team Workflow
1. Create a research draft branch
2. Add or update research materials
3. Commit changes
4. Push branch
5. Create Pull Request
6. Obtain review approval
7. Merge into `main`
